# VPN DevOps Project

## Description
Personal VPN service based on WireGuard with DevOps practices.

## Stack
- Linux (Ubuntu)
- WireGuard
- Docker
- Kubernetes (k3s)
- Ansible
- Zabbix + Grafana
- GitHub Actions

## Architecture
- VPS-1: Control & Monitoring
- VPS-2: Kubernetes Master
- VPS-3: Kubernetes Worker + VPN
